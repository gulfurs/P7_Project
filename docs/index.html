Adaptive Unity Game System

Introduction

Can an local LLM-driven multi-agent system that facilitates natural conversation be perceived as natural and attentive in a multi-party interview setting while maintaining acceptable latency constraints?

Using Local LLM, the agents response is dynamically generated and adaptive to the users input. The NVB animationsis also adaptive according to their response

To adress conversational AI latency and LLM social blindness in multi-part dynamics

Problem Definition

Original Problem: Local pipeline of STT → LLM → TTS with dynamic avatars, trying not to break conversational grounding

Requirements: Under x00 end-to-end latency; context-aware avatars + NVB

Constraints: Current State of the art, STT speed-accuracy tradeoffs; LLM understanding of human dynamics; TTS expressiveness / latency,
